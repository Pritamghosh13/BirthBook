# 🎂 BirthBook

BirthBook is a web application that helps users store and track important birthdays, with automatic reminders and countdowns to upcoming dates. It ensures you never miss special occasions through simple and efficient date management.

---

## 🚀 Features

- ➕ Add and manage birthdays
- 📅 Automatic countdown to upcoming birthdays
- 🔔 Birthday reminders
- ✏️ Update and delete entries
- 👤 User-based data management
- ⚡ Clean and efficient backend logic

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Other Tools:** REST API, Date handling
